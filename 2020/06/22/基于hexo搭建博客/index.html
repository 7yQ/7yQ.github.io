基于hexo搭建博客
发表于 2020-06-22| 更新于 2020-06-23
阅读量:

第一步：

下载 node.js 并安装 , 下载网址为: https://nodejs.org/en/

第二步：

进root,命令： sudo su

下载cnpm 命令为: npm install -g cnpm –registry=https://registry.cnpm.taobao.org

第三步：

创建文件夹并初始化

mkdir blog

sudo hexo init

启动博客 hexo s

第四步：

部署git

cnpm install –save hexo-deployer-git

修改_config.yml

type： git
repo: github仓库http地址

第五步：

创建新的md文件

hexo n ‘博客名’

第六步：

清空缓存

hexo clean

提交文件

hexo g

上传gtihub

hexo d

修改主题

git clone https://github.com/yelog/hexo-theme-3-hexo.git themes/主题

修改_config.yml

themes: 主题
